## Projects

### [Weather DJ](https://github.com/Gyiben/Weather_DJ)
A Streamlit app that curates Spotify playlists based on real-time weather in your city. Integrates the Spotify API and OpenWeather API with animated weather visualizations.

### [HR Scanner](https://github.com/Gyiben/HR_scanner)
A design fiction prototype that exposes algorithmic bias in hiring. Simulates a corporate screening tool that scores applicants by analyzing social media, then critiques its own outputs to show how polished dashboards lend false credibility to biased predictions.
